# Travel Tracker 🌍

A full stack web app to track visited countries on an interactive world map.

## Tech Stack
- Node.js
- Express.js
- PostgreSQL
- EJS (Templating Engine)

## Features
- Add visited countries
- Interactive world map
- Real time country count

## Setup
1. Clone the repo
2. Run `npm install`
3. Setup PostgreSQL database
4. Add `.env` file
5. Run `npm start`
